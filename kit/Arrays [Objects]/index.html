Object Propriétés

On peut accéder aux propriétés d'un objet dans un tableau de cette manière : array[index].propertyName

const cart= [{
  color: 'Green',
  
},
{
  color: 'red',
  
},
{
  color: 'blue',
  
}
];
console.log(cart[0].color); // 'Green'
console.log(cart[1].color); // 'red'


Parcourir un tableau

Il existe plusieurs méthodes pour parcourir un tableau, Map (), for , while etc.. Voici un exemple avec la boucle for : Plus d'info

const students =[ 
{
  name: 'Sheldon',
  score: 100,
  grade: 'A',
},
{
  name: 'Ahmed',
  score: 10,
  grade: 'F',
},

{
  name: 'Fred',
  score: 110,
  grade: 'A+',
},

]

for (const element of students ) {
console.log(element.name); }

Map()

map La méthode map(), Permet de parcourir un tableau, de faire un traitement particulier puis crée un nouveau tableau
Plus d'infos
