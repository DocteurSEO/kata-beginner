Syntaxe

Les tableaux sont des objets semblables à des listes dont le prototype possède des méthodes qui permettent de parcourir et de modifier le tableau. 

const numbers = [1, 2, 3, 4];

Index

Les éléments du tableau sont disposés par index valeurs, à partir de 0 comme premier élément. Les éléments sont accessibles par leur index en utilisant le nom du tableau et l'indice entouré de crochets. 

// Accessing an array element
const myArray = [100, 200, 300];
 
console.log(myArray[0]); // 100
console.log(myArray[1]); // 200
console.log(myArray[2]); // 300

Method .length

.length La propriété length (longueur) est un entier qui indique le nombre d'éléments présents dans le tableau. 

const cart = ['apple', 'orange'];
cart.push('pear'); 
 

cart.length  // 3

Method .push()

.push() peut être utilisée pour ajouter un ou plusieurs éléments à la fin d'un tableau. 

// Adding a single element:
const cart = ['apple', 'orange'];
cart.push('pear'); 
 
// Adding multiple elements:
const numbers = [1, 2];
numbers.push(3, 4, 5);

Method .pop()

.pop() supprime le dernier élément d'un tableau et retourne cet élément. 

const ingredients = ['eggs', 'flour', 'chocolate'];
 
const poppedIngredient = ingredients.pop(); // 'chocolate'
console.log(ingredients); // ['eggs', 'flour']

Mutable

Les tableaux JavaScript sont mutable, ce qui signifie que les valeurs qu'ils contiennent peuvent être modifiées.

Même s'ils sont déclarés en utilisant const, le contenu peut être manipulé en réassignant des valeurs internes ou en utilisant des méthodes comme .push() et .pop(). 

const names = ['Alice', 'Bob'];
 
names.push('Carl');
// ['Alice', 'Bob', 'Carl']

Arrays

Les tableaux sont des listes de données ordonnées et données. Ils peuvent contenir des éléments de n'importe quel type de données. Les tableaux sont créés à l'aide de crochets, les éléments individuels étant séparés par des par des virgules.

// An array containing numbers
const numberArray = [0, 1, 2, 3];
 
// An array containing different data types
const mixedArray = [1, 'chicken', false];
