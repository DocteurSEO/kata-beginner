Syntaxe

Maintenant, JavaScript intègre un moyen qui lui est propre de créer des requêtes API. Il s’agit de l’API Fetch, une nouvelle norme qui permet de réaliser des requêtes sur un serveur en utilisant des promesses. Elle intègre cependant bien d’autres fonctionnalités.

fetch(url) .then(function() { }) .catch(function() {});

Comment passer le Bearer Token dans la requête fetch ?



Vous pouvez passer des en-têtes HTTP à la requête fetch() comme deuxième paramètre. Par exemple, pour passer l'en-tête d'autorisation Bearer Token, appelez fetch() avec le paramètre {headers : {Authentication : 'Bearer Token'}}
. L'exemple ci-dessous montre comment envoyer une en-tête HTTP personnalisé

fetch("https://seo-fast-audit.p.rapidapi.com/scraper?url=https%3A%2F%2Feasy-video.fr%2F&selector=h1", "method": "GET"
"headers": {
"x-rapidapi-host": "seo-fast-audit.p.rapidapi.com",
"x-rapidapi-key": "b5b2ec9656msh285edf6f388e087p1f0c5ajsn3b5bb3db591c" } })
.then(response => { console.log(response); })
.catch(err => { console.error(err); });
